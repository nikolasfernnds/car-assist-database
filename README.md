# Car-Assist Database

![Car-Assist Logo](assets/image.png)

Este repositório contém o banco de dados do projeto **Car-Assist**, incluindo modelos conceituais, lógicos e scripts de criação e populamento de dados.

This repository contains the **Car-Assist** project database, including conceptual and logical models and SQL scripts for database creation and data population.

## Folder Description

- **models/conceptual**: Conceptual model of the database  
- **models/logical**: Logical model of the database  
- **scripts**: SQL scripts for creation, insertion, and maintenance of the database
